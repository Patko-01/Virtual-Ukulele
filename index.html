Help - notes

Some notes are located on multiple strings, or the note can be played on one string in multiple places. This is why in some cases it's not enough to type in just the name of the note, you also need to type in the fret on which is the note located.

Example for note "G#" on the first fret:

G#-1

(-1 stands for first fret).
